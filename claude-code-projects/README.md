# Claude Code Projects

A collection of projects built using Claude Code — some security-relevant, some just things I thought would be fun to build. The honest reason most of these exist is to get my reps in. The best way to get comfortable building things is to actually build things, and Claude Code makes it possible to ship ideas fast without getting stuck on boilerplate.

[![Status](https://img.shields.io/badge/Status-Growing-grey?style=flat)]()

## Why Claude Code

I'm actively working through Anthropic's courses on AI fluency, prompt engineering, and the Claude API — and Claude Code is the natural place to put that learning into practice. There's a difference between understanding how to work with AI tools and actually using them to build something from scratch. This folder is where that gap gets closed.

Some projects here will be security-focused. Others will just be ideas I wanted to see work. Either way the goal is the same — build, break, learn, repeat.

## How These Are Built

Every project in this folder was built collaboratively with Claude Code. Each one includes a short write-up covering what the project does, why I built it, what I learned from the process, and anything interesting that came up along the way — bugs, unexpected behavior, design decisions.

The write-ups are as important as the code. Building something and understanding what you built are two different things.
