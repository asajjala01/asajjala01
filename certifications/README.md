## My Approach to Documenting Learning

Every folder here follows the same format — course or module breakdown, key concepts, and a "My Take" section at the end of each one. The personal notes aren't an afterthought, they're the point. Anyone can copy notes from a course. What matters is whether you actually internalized it and can connect it to real work.

I also try to be honest about what was hard. Not everything clicked immediately and pretending otherwise doesn't help anyone — including me looking back at these notes six months from now.

As new certifications are completed they'll be added here with the same level of documentation. This folder will keep growing.
